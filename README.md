
📚 StudyLend
On-chain student savings pools & microloans (SEA)
🧠 Overview

StudyLend is a decentralized savings and lending platform built for student organizations in Southeast Asia.

It allows students to:

💰 Pool funds (paluwagan-style)
⚡ Access instant microloans
🤝 Avoid informal lenders and social friction
🚨 Problem

Students often face short-term financial gaps, especially before critical academic events.

📍 Real Scenario

A nursing student in Cebu needs ₱2,000 for review materials two weeks before board exams.

⚠️ Current Options
❌ No stipend yet
❌ Informal lenders charge 20% weekly interest
❌ Borrowing from friends → no structure + broken trust
💡 Solution

StudyLend introduces on-chain student savings pools powered by Soroban smart contracts.

🏗️ How It Works
Student organizations create shared pools
Members deposit USDC
Members request loans anytime
🔐 Smart Contract Enforces
Loan caps
Interest rates
Repayment rules
✅ Benefits
No collateral
Fully transparent
No IOUs or social tension
⚙️ MVP Transaction Flow
1. Treasurer creates pool (max cap + interest)
2. Members deposit USDC into the contract
3. Student requests loan (request_loan)
4. USDC is disbursed instantly (<5s)
5. Student repays with interest
6. Pool grows for future borrowers
🔗 Key Features
🏦 On-chain Paluwagan

Shared savings pool managed by smart contract

⚡ Instant Microloans

No approvals, no waiting

📜 Smart Contract Enforcement

Trustless system — rules are automatic

💸 Low Interest

~2% flat vs informal lenders

🔍 Transparent Ledger

All transactions are verifiable on-chain
